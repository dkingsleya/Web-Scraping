# py-web-scrape
Web Scraping Trading View to Get First 150 Coins Data
With Tkinter GUI

## Main Libraries
######Beautiful Soup 4
> pip install beautifulsoup4

######Requests
> pip install requests

######LXML
> pip install lxml

######Tkinter
> pip install tkinter

## Pyinstaller Commands

```
pyinstaller --onefile -w 'scraping.py'
pyinstaller scraping.spec

```
